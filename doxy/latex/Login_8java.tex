\doxysubsection{Detailed Description}
Interfaz gráfica para el inicio de sesión de usuarios. 

Esta clase proporciona una ventana Swing sencilla que permite al usuario introducir un nombre de usuario, una contraseña y seleccionar el tipo de usuario (admin o cashier). Según las credenciales y el tipo seleccionado, la aplicación abre la ventana correspondiente (Admin\+Login o Cashier).
